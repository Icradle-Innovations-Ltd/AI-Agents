## Troubleshooting

### "API Key Invalid"
- Check `.env` file has the correct key
- Make sure no extra spaces or quotes
- Test at https://platform.openai.com/account/api-keys

### "Rate Limited"
- Alpha Vantage limits to 5/minute, 500/day
- NewsAPI limits to 450/day
- Add delay between requests (built-in)
- Try again after 60 seconds

### "Out of Quota"
- Check https://platform.openai.com/account/billing
- Add payment method if needed
- Set usage limits to prevent surprise charges

### "Real Stock Data Would Be Better"
- Get Alpha Vantage key (free 500/day)
- Add to `.env`: `ALPHAVANTAGE_API_KEY=your_key`
- Workflow automatically uses it
