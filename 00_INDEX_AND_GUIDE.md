## 💡 Key Concepts Across All Documents

### The Central Insight (Appears in all documents)
```
Workflow → Agent transformation happens when:

HUMAN DECISION-MAKER → LLM DECISION-MAKER

It's not about tools. It's not about steps. 
It's about who decides how to solve the problem.
```

### The Three Levels (Simplified)
```
Level 1 - LLM:      Human Input → AI Output
Level 2 - Workflow: Human Programs → AI Follows → Output  
Level 3 - Agent:    Goal → AI Reasons/Acts/Iterates → Output
```

### ReAct Framework (The Agent Pattern)
```
Reason → Act → Observe → Iterate? → Return Output
```
